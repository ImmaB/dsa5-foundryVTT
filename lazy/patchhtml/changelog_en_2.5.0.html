2.5.0 Important new features (12/07/2021)

Foundry 9

This patch does not provide a lot of new features (there were already plenty last month). Instead the focus is on stability and bugfixes because the next version of Foundry is almost finished. We will now focus on the required changes for the update. Please refrain from using TDE 2.5.0 with Foundry 9 because it is not yet compatible. We'll provide a beta soon and with the arrival of the beta support and updates for Foundry 0.8 and TDE will stop.

Character calculator

The core module now contains a calculator which can be used to assist in the character generation. It sums up Adventure Point costs of several character elements and attribute sums dependent on character experience and similar things. You can find it by typing /cc in the chat or using the macro in the core macros compendium. Drag your actor onto the calculator to see your AP costs (Players need to drag the token link next to the name on their character sheet). Please refrain from reporting any tickets concerning character generation or this tool. We do not plan to deal with the pandora's box of character generation. Tickets will be ignored, cut into four, drowned and redirected to the nameless one.

Enhancements

  • The status defeated is now available via the character sheet and token HUD (instead of only the combat tracker)
  • Spells now show a check mark for paid costs too
  • Applied damage on opposed rolls is now visible for players too
  • Actors can now be dragged and dropped via a link next to the name on the actor sheet
  • The library can now filter by modules too
  • Aspect knowledge/Aspektkenntnis now calculates max advancements for liturgies
  • The damage tooltip for opposed rolls is now a little bit better
  • Regeneration and attribute blind rolls (hidden to players) can now be done via the game masters menu
  • RQ rolls in the chat now accept + in front of the modifier too
  • Breaking point rolls now show a warning if the weapon is blessed or magical (because it might be indestructible)

Bugfixes

  • The Species wizard now properly shows descriptions of custom species
  • Unarmed combat now rolls damage properly again
  • Soundeffects are not played for secret rolls anymore
  • Repeat damage roll should now be calculated properly
  • Rangeweapons and Meleeweapons can now contain effects similar to combat special abilities and spells (you can create a flamesword now)
  • Modifiers for regeneration request rolls should now be available
  • The master's workshop should be more compatible with The Forge now. Forge modifies the module which leads to some problems. You will still have to manually set the path to tokens.
