3.0.1 Important new features (12/20/2021)

Foundry 9

The Dark Eye is now compatible with Foundry 9. All modules should also work with the new version. Their versions will be adjusted at a later point in time.

Improvements

  • Added a few keybinds
  • Journals linked on the current map now have a pin button to jump directly to the link (especially useful for RSHs).
  • Combat Techniques can now also be used for opposed tests.

Bugfixes

  • A bug with creature types and magical and blessed weapons has been fixed
  • Darksight 2 should now be correctly included in the view automation.
  • Items and status effects can no longer be deleted from locked compendia.
  • A bug with the calculation of the duration of effects has been fixed.
